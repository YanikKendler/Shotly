# Templates

Templates can be selected when creating a shotlist so that you don't have to create the same attributes over and over again.

Changes made to a template will **not** be reflected in existing shotlists. Every shotlist manages its own attributes. Only those created based on this template *after* it has been edited will use the updated attributes.

This also means that deleting a template has no effect on any shotlists. Additionally, there is no way for the template owner to access or edit your shotlist if you created it using a shared template.

## Editing

Templates can be edited by selecting a template from the sidebar or dashboard. 

New "attribute templates" can be added using the `Add scene attribute` / `Add shot attribute` buttons. These reflect the [Attributes](./shotlist/attributes.md) that will later be created in the shotlist.

A template can be deleted by clicking `Delete template` in the top right.
