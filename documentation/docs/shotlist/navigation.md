# Navigation

The shotlist is broken up into Scenes, which are located in the sidebar. As a scene is selected, its shots are loaded and displayed in the table on the right.

At the bottom of the sidebar the button `Collab-Requests` is replaced with the `Shotlist Options` button. This button opens a dialog that allows renaming the shotlist, [editing the attributes](./attributes.md), [managing collaborators](./collaboration.md) and [filtering and exporting the shotlist](./export.md). 

!!! Tip
    The shotlist can also be renamed by simply clicking the shotlist's name in the top left next to the dashboard icon.

Both shots and scenes can be modified and reordered using the "6 dots grip-icon" that appears when hovering either. By clicking and dragging this grip, the scene/shot can be moved and re-ordered. By clicking on the grip a context menu appears that allows you to move, edit or delete a scene/shot.
