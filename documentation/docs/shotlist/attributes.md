# Attributes

The attributes for all scenes and shots of a given shotlist can be controlled in one central location: `Shotlist options` > `Attributes`.

Currently, Shotly supports three types of attributes:

- **Text**: a standard text field that stretches multiple lines.
- **Single select**: a searchable drop-down that allows the selection of a single value.
- **Multi select**: a searchable drop-down that allows the selection of multiple values.

!!! Tip
    The **attribute menu** can also be opened from the single- and multi-select **dropdowns**, as well as through the shot and scene **context menu** that appears when clicking the six dots next to a shot or scene.

## Manage Attributes

When clicking the `Add attribute` button, a pop-up menu will appear, and you can choose the **attribute type** to add.

Each attribute can be **deleted** by using the trash bin icon to the right. A dialog will appear that asks for your confirmation. Deleting a column will **permanently remove** the selected attribute from the shotlist. This means that **all the data in that column** of the list or that attribute of a scene will be **lost**.

Attributes can be **reordered** by using the drag handle to the left of each attribute in the attribute dialog or by using the buttons in the **context menu**.

The options for single- and multi-select attributes can be **edited** either through the `Edit options` button or by typing a new value into the dropdown and selecting `create "..."`.

!!! Info
    Attributes can be edited by all [Collaborators](./collaboration.md) with the "Editor" role.
